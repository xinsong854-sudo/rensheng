- 🎨 精美立绘 - 高质量 + 画师/风格 + 发型 + 服装 + 光影

### 技术细节

- 装饰图片：`decorative-image.jpg`，最大宽度 600px，圆角 12px
- 标签格式：`tag.en · tag.cn`
- 敏感词分级：高置信度（红）/ 可能误报（黄）
- FAQ：折叠/手风琴形式
- 随机生成：从所有标签分类中抽取 2-5 个元素
