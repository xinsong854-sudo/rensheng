## 🎨 设计系统

### 颜色变量
```css
--primary-color: #ff6b35       /* 主色调 */
--primary-hover: #ff8c5a       /* 悬停色 */
--primary-light: rgba(255, 107, 53, 0.08)  /* 浅色背景 */
--background: #faf8f5          /* 页面背景 */
--text-primary: #333333        /* 主文字 */
--text-secondary: #555555      /* 次要文字 */
--text-light: #888888          /* 辅助文字 */
```

### 圆角规范
```css
--radius-sm: 8px    /* 小按钮/标签 */
--radius-md: 12px   /* 卡片/按钮 */
--radius-lg: 16px   /* 大卡片 */
--radius-xl: 20px   /* 内容区域 */
```

### 阴影规范
```css
--card-shadow: 0 4px 16px rgba(0, 0, 0, 0.06)      /* 常规卡片 */
--card-shadow-hover: 0 8px 24px rgba(0, 0, 0, 0.1) /* 悬停状态 */
```

## 🚀 使用建议

### 部署新详情页
1. 复制 `detail-template.html` 为具体书籍页面
2. 替换 `BOOK_DATA` 中的书籍信息
3. 更新章节列表和相关推荐

### 部署新章节
1. 复制 `reading-template.html`
2. 替换模板变量：`{{CHAPTER_TITLE}}`, `{{CONTENT}}` 等
3. 更新 `chapters` 数组和 `currentChapterIndex`

### 主题定制
- 修改 CSS 变量即可全局换色
- 夜间模式自动适配所有组件
- 字体大小用户可自定义并保存
