# 书籍封面显示修复报告

## 问题描述
在分类页面（如 books/characters/）点进去看书籍列表时没有封面显示。

## 修复内容

### 1. 修复 shujia/index.html（书架主页）
- ✅ 更新书籍数据，将 `emoji` 和 `color` 属性替换为 `cover` 属性
- ✅ 修改 `renderBookshelf()` 函数，使用 `<img src="${book.cover}">` 显示封面
- ✅ 修改 `renderRecommendations()` 函数，使用封面图片显示
- ✅ 修改 `toggleFavoriteFromIndex()` 函数，使用 `cover` 属性存储收藏数据
- ✅ 添加图片加载失败回退：`onerror="this.style.display='none';this.parentElement.innerHTML='📚'"`

### 2. 修复 shujia/books/index.html（分类索引页）
- ✅ 更新 CSS 样式，添加 `.card-cover` 类用于封面显示
- ✅ 修改所有分类卡片（人物志、伪人录、里界探秘、组织档案）
- ✅ 使用正确路径 `../covers/xxx.jpg` 引用封面图片
- ✅ 保留收藏功能（data-book 属性和 favorite-btn 按钮）
- ✅ 添加图片加载失败回退显示 emoji

### 3. 封面路径说明
```
shujia/index.html          → 使用 ./covers/xxx.jpg
shujia/books/index.html    → 使用 ../covers/xxx.jpg
shujia/books/characters/   → 使用 ../../covers/xxx.jpg
```
