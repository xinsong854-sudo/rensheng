🔑 捏 Ta Token 获取工具

手机端专用 · 无需电脑 · 一键获取

1

点击下面按钮加载调试工具

会自动打开捏 Ta 官网，并在页面上显示一个 🐞 调试按钮

💡 打开后，页面右下角会出现一个半透明的 🐞 按钮，点击它就能打开控制台
2

在捏 Ta 页面获取 Token

1. 点击右下角的 🐞 按钮

2. 点击 Console 标签

3. 粘贴下面的代码并运行

// 复制这段代码运行 const token = localStorage.getItem('token') || sessionStorage.getItem('token') || document.cookie.match(/token=([^;]+)/)?.[1]; if (token) { console.log('✅ Token 获取成功！'); console.log('Token:', token); // 自动复制到剪贴板（部分浏览器支持） if (navigator.clipboard) { navigator.clipboard.writeText(token); console.log('📋 已复制到剪贴板！'); } } else { console.log('❌ 未找到 Token，请确认已登录'); }
💡 运行后，Token 会显示在控制台，并且会自动复制
3

把 Token 粘贴到开盒工具

回到开盒工具页面，粘贴到"自定义 Token"输入框
