🔧 捏 Ta 调试工具

手机端专用 · 自动注入控制台

📱 选择你的浏览器

1
下载并安装 Kiwi Browser
2
安装捏 Ta 调试扩展

打开 Kiwi → 右上角菜单 → Extensions → + (from .zip)

下载后在 Kiwi 中解压并安装
3
打开捏 Ta
扩展会自动注入 Eruda 调试工具！
1
复制下面的代码
javascript:(function(){var s=document.createElement('script');s.src='https://cdn.jsdelivr.net/npm/eruda';s.onload=function(){eruda.init();eruda.show();};document.body.appendChild(s);})();
2
打开捏 Ta
3
在捏 Ta 页面运行代码

长按地址栏 → 粘贴 → 点击运行（Go/前往）

💡 代码以 javascript: 开头，运行后地址栏不会跳转，这是正常的！

🔑 获取 Token

1

点击右下角 🐞 按钮 → Console 标签

2

粘贴并运行下面的代码：

const token = localStorage.getItem('token') || sessionStorage.getItem('token') || document.cookie.match(/token=([^;]+)/)?.[1]; if (token) { console.log('✅ Token:', token); navigator.clipboard.writeText(token); console.log('📋 已复制！'); } else { console.log('❌ 未找到 Token'); }
