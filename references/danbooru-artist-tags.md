## 使用示例

### 单画师
```
artist_lack, detailed, realistic, dramatic_lighting
artist_huke, dark, mechanical, black_rock_shooter
artist_artgerm, dc_comics, marvel, superhero
artist_wlop, fantasy, ethereal, ghostblade
```

### 多画师混合
```
artist_lack OR artist_wengwengchim OR artist_imamiya_pinoko, realistic, thick_paint
artist_huke OR artist_redjuice OR artist_ogura, semi_realistic, anime
studio_ghibli OR kyoto_animation, anime, cel_shading, soft_colors
```

### 风格组合
```
# 厚涂写实
artist_lack, realistic, detailed, cinematic_lighting

# 赛璐璐动画
studio_ghibli, anime, cel_shading, fantasy, adventure

# 概念艺术
artist_syd_mead, sci-fi, concept_art, futuristic, cyberpunk

# 美漫超级英雄
artist_artgerm OR artist_alex_ross, marvel, dc_comics, superhero, action

# 数字插画
artist_sakimichan OR artist_krenz, digital_art, thick_paint, detailed

# Q 版可爱
chibi, kawaii, super_deformed, cute, pastel_colors

# 水墨和风
ink_wash_painting, sumi_e, japanese_style, traditional, monochrome
```

## 注意事项

1. **标签格式：** 画师使用 `artist_名字` 格式，工作室/公司直接使用名字
2. **下划线：** 所有空格用下划线 `_` 替代
3. **大小写：** 标签通常小写，但部分画师名保留大写
4. **特殊字符：** 避免使用特殊字符，用下划线替代
